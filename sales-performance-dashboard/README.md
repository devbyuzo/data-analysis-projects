# Sales Performance Dashboard (Excel)

## 📌 Overview
This project analyzes sales data to track performance across products, regions, and time periods using an Excel dashboard.

---

## 🎯 Objective
To transform raw sales data into actionable insights that help identify trends, top-performing products, and revenue patterns.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Data Cleaning Functions

---

## 📊 Key Features
- Monthly sales trend analysis
- Top-performing products identification
- Regional sales comparison
- Revenue summary dashboard

---

## 🔍 Key Insights
- Certain products consistently outperform others across all regions
- Sales show clear peak periods within the year
- Some regions contribute significantly more to total revenue
